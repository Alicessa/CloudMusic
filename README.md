# netcloud
# 项目预览地址http://112.74.78.149/netmusic

## 这是一个仿网易云音乐的项目目前暂不支持列表播放及登录效果，

##后续将会有可能加入这些功能，如果这个项目对你一有所帮助请你们点亮星星😁😁😁❤
```
项目开始的操作
npm install
```

### 项目启动
```
npm run serve
```

### 打包操作

npm run build

## 目录清晰，全部的组件没有抽离，都一一暴露在src/components下
